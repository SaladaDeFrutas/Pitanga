3.1 Procedimento para revisões

a) Fazer a rastreabilidade dos requisitos da Solicitação de Mudança ou Relatório de Problema para o design e código.

b) Verificar a testabilidade do código.

c) Verificar a conformidade com padrões de código.

d) Verificar se somente o necessário foi modificado.

e) Checar se a documentação necessário foi atualizada.

f) Desenvolver o relatório de revisão seguindo o template.
